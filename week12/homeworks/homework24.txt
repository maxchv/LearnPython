1) Создать проект academy для автоматизацию учета курсов
2) Добавить приложение teachers с моделью Teacher:

	Teacher
	- имя	(first_name)
	- фамилия	(last_name)
	- эл.почта	(email)
	- телефон	(phone)
	- дата рождения (date_of_birth)
	- пользователь (FK = связь один-к-одному на User)

3) Добавить приложение courses с моделью Course:  
	
	Course
	- название (name)
	- описание (description)
	- преподаватель (teacher - FK = связь один-ко-многим на Teacher)	
	- начало курса (start)
	- окончание курса (end)

4) Добавить прложение students с моделью Student:
	
	Student
	- имя  (first_name)
	- фамилия (last_name)
	- эл.почта (email)
	- телефон (phone)
	- ссылка на страницу в соцсети (url - может быть пустой)
	- дата рождения (date_of_birth)
	- курсы (courses - M2M = связь многие-ко-многим на Course)
	
4) Зарегестрировать созданные модели в модуле admin.py
5) Создать администратора сайта с логином admin и паролем python2016
6) Заполнить базу данных используя административную часть сайта
